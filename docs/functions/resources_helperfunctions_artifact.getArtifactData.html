Function getArtifactData

  • getArtifactData(artifactName: string, destination: string, artifactObject: Artifact): Promise<DownloadResponse>

  • Downloads an artifact that was uploaded by another action in a previous step or job in the workflow.

    Returns

    A DownloadResponse object containing the download path and the artifact name.

    Parameters

    • artifactName: string

      The name of the artifact given by the action that created it.

    • destination: string

      The directory in which the artifact files should be downloaded.

    • artifactObject: Artifact

      The Artifact module that is used. During normal operation of the program, this should simply be @actions/artifact, but for the unit tests a mock is passed instead.
